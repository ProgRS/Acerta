## 🧲 Como baixar o projeto

```bash
# Clonar o repositório
$ git clone https://github.com/ProgRS/Acerta
# Entrar no diretório 
$ cd Acerta
# Instalar as dependências
$ npm install
# Rodar a api fake
$ npx json-server --watch -p 3333 acertaleads
# Iniciar o projeto
$ npm start
